# Weekly Review Template

Last updated: 2026-03-29

## Purpose

One document to fill out every week that pulls together all metrics,
answers the review questions, and produces a clear set of next-week
decisions. Copy this template into a new file each week (e.g.
`reviews/week-01.md`) or use it as a recurring spreadsheet tab.

Complete this every Monday morning before starting the week's outbound.

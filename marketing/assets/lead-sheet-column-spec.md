# Lead Sheet Column Spec

Use these columns in the spreadsheet-first CRM.

| Column | Purpose |
| --- | --- |
| created_at | first timestamp in system |
| company_name | business name |
| contact_name | primary person if known |
| city | target city |
| trade | target trade |
| website | current website |
| phone | best phone number |
| email | best email or contact path |
| lead_source | source taxonomy value |
| campaign | campaign or UTM campaign |
| partner_name | partner name if referred |
| score | outbound or fit score |
| key_issue | most visible website problem |
| notes | supporting context |
| stage | current pipeline stage |
| owner | who owns the next action |
| last_contact_at | last human touch date |
| next_action_at | next required action date |
| quote_value_estimate | expected deal size |
| outcome | won, lost, nurture, open |

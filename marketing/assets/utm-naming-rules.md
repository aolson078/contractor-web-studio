# UTM Naming Rules

## Required Parameters

- `utm_source`
- `utm_medium`
- `utm_campaign`

Use `utm_content` when testing variants.

## Naming Standard

- lowercase only
- use underscores, not spaces
- keep names human-readable

## Source Examples

- organic_search
- cold_email
- partner_referral
- linkedin
- blog
- gbp

## Medium Examples

- email
- referral
- social
- organic
- profile

## Campaign Examples

- q2_masonry_outbound
- greensboro_city_page
- wix_comparison
- photographer_partner_push

## Example URL

`https://contractorwebstudio.com/contact?utm_source=cold_email&utm_medium=email&utm_campaign=q2_masonry_outbound&utm_content=email_1`
